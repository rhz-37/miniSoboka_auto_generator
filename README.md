Sokoban           

主菜单（Menu）

  [Easy]    简单模式
  [Medium]  中等模式
  [Hard]    困难模式
  [Hell]    地狱模式
  [Records] 查看通关记录
  [3D!]     第一人称 3D 模式
  ·鼠标点击按钮
  ·键盘 ↑↓ + Enter 选中
  ·键盘 1~5 快捷选择（1=Easy ~ 5=Records）

游戏界面（2D）

  白色方块 = 墙壁
  绿色方块 = 箱子
  红色标记 = 目标点
  蓝色圆点 = 玩家

    [CCW]          逆时针旋转地图
    [CW]           顺时针旋转地图
    [Undo]         撤回上一步
    [Restart]      重新开始本关
    （通关后）
    [Next]          下一关（同难度随机）
    [Replay]        重玩本关
    [Menu]          返回主菜单

3D 

    W          朝视角方向前进
    S          朝视角反方向后退
    A          朝视角左侧平移
    D          朝视角右侧平移
    Z          撤回上一步推箱
    ESC / M    返回主菜单
    按住左键拖动 → 转动视角（水平 + 垂直）
    推箱：走到箱子前，朝箱子方向前进即可推动

记录页面（Records）

    [Replay (R)]     重放选中的记录（完整回放操作序列）
    [Del]            删除选中的记录
    [Back (ESC/M)]   返回主菜单
    ↑/↓        上下选择记录
    R          重放选中的记录
    Delete     删除选中的记录
    ESC / M    返回主菜单

回放页面（Replaying）

    [CCW]        逆时针旋转地图
    [CW]         顺时针旋转地图
    [Replay]     从头开始重放
    [pos_solu]   运行求解器找出最优解并重放（≤4箱可用）
    [Play Live]  从当前位置切换为手动操作
    [Menu]       返回主菜单
